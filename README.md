# Sravanth Kumar Portfolio

This is a personal portfolio website showcasing my skills, projects, coding profiles, and contact information as a Java Developer and Computer Science student.

## Project Structure
- **index.html**: Main HTML file for the portfolio structure.
- **assets/css/style.css**: Custom CSS with variables and responsive design.
- **assets/js/script.js**: JavaScript for interactivity (sidebar, navigation, project filtering, form validation).
- **assets/images/**: Contains favicon, skill icons, project thumbnails, and coding platform logos.

## Dependencies
- **Google Fonts**: Poppins font (loaded via CDN).
- **Ionicons**: Icons for social links, contact info, and buttons (loaded via CDN).
- **Formspree**: Contact form submission endpoint (https://formspree.io/f/xjkbwjlj).

## Setup Instructions
1. **Clone or Download**: Download `Sravanth_Kumar_Portfolio.zip` and extract it, or clone from [https://github.com/sravanthskr/portfolio](https://github.com/sravanthskr/portfolio).
2. **Serve Locally**:
   - Use a local server (e.g., `live-server` in VS Code or `python -m http.server 8000`).
   - Open `http://localhost:8000` in a browser.
3. **Verify Images**: Ensure all files in `assets/images/` are present and correctly named.
4. **Formspree**: The contact form submits to `https://formspree.io/f/xjkbwjlj`. Ensure the endpoint is active or replace with your own.
5. **GitHub Link**: Update the GitHub social link in `index.html` (line 135) to `https://github.com/sravanthskr`.

## Features
- **Responsive Design**: Adapts to mobile, tablet, and desktop screens.
- **Interactive Sidebar**: Toggles on mobile to show contact and social links.
- **Project Filtering**: Filter projects by category (Java, AI ML, Web Development, UI UX, Flutter).
- **Contact Form**: Validates input and submits via Formspree.
- **Coding Profiles**: Links to LeetCode, Coding Ninjas, HackerRank, and CodeChef profiles.
